streamdefence changes iframes to encrypted with streamdefence.com API

== Description ==

This plugin allows us to use the services offered by streamdefence.com via API
StreamDefence protects your videos from bots & Reports / Removal and block who would "steal" and "repost" your third-party stream plus Automatically regenerate your Openload.co videos if are deleted.
The video player of third party sites such as Openload.co, Youtube.com, Dailymotion.com, Primevideos.net, Vidlox.me, Streamango.com, Vidoza.net and many others, will be encrypted and protected so that malicious people can see the iframe of the video and copy it.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Streamdefece settings screen to configure the plugin
4. Enter API Code generated by Streamdefence.com
5. Enter your security options,Click on Save
Each video in a new published or edited post will be automatically encrypted.
6. For the correct functioning of the plugin for wordpress versions 5 and higher, it is advisable to install the Disable Gutenberg plugin
